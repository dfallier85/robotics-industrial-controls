# Operator and HMI

## Main point

The operator should be able to understand and influence the cell without needing to read PLC or robot code.

## Typical HMI functions

- start
- stop
- reset
- mode selection
- alarm display
- status display
- manual jog or maintenance screens
- recipe or part selection

## Why this matters

The HMI is the human-facing explanation of the cell.

If the architecture ignores the operator layer, it ignores a major part of how the machine is actually used.
