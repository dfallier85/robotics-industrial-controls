# System Layers

## Main idea

An industrial robot cell usually contains several layers working together.

## Common layers

### 1. Operator layer
- HMI
- buttons
- indicators
- status screens

### 2. Supervisory / sequence layer
- PLC logic
- machine states
- permissives
- command routing

### 3. Robot execution layer
- robot controller
- programs
- motion execution
- EOAT actions

### 4. Perception / sensing layer
- vision
- photoeyes
- presence sensors
- position feedback

### 5. Physical machine layer
- conveyors
- fixtures
- tooling
- pneumatics
- actuators

### 6. Safety layer
- e-stops
- gates
- interlocks
- safety relays / safety PLC functions

## Why this matters

You troubleshoot and design cells better when you know which layer owns which behavior.
