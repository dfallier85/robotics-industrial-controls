# Safety Fault Checklist

## 1. Initial fault capture
- [ ] Record exact fault text
- [ ] Record alarm code if shown
- [ ] Record date / time
- [ ] Record station / robot
- [ ] Note whether issue is steady or intermittent
- [ ] Note recent shutdown or maintenance history

## 2. Physical device checks
- [ ] Inspect door switch / sensor alignment
- [ ] Inspect E-stop condition
- [ ] Inspect reset button condition
- [ ] Test device continuity or voltage change
- [ ] Confirm device changes state correctly

## 3. Wiring path checks
- [ ] Check terminal block connections
- [ ] Check wire damage or loose terminations
- [ ] Confirm signal reaches relay/module input
- [ ] Confirm relay/module output changes
- [ ] Confirm signal reaches controller input

## 4. Relay / safety hardware checks
- [ ] Measure relay coil voltage
- [ ] Verify contact transfer behavior
- [ ] Confirm NO/NC expectations
- [ ] Check safety module indicators if present

## 5. Controller / logic checks
- [ ] View controller input state
- [ ] View PLC input tag if applicable
- [ ] Check HMI reset bit behavior
- [ ] Check sequence step / interlock conditions
- [ ] Confirm fault acknowledgment path

## 6. Conclusion
- [ ] Identify what was ruled out
- [ ] Identify most likely fault location
- [ ] State recommended next action
- [ ] Document whether escalation is needed
