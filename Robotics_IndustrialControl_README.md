# ⭐ Flagship System

## Gesture-Controlled UR System

A full-stack robotic control system integrating:

- 🎥 Vision-based input (gesture recognition)
- 🤖 RTDE robot motion control
- 🛡 Safety gating and deadman logic
- 📦 Workspace limiting and motion inhibition
- ⏱ Watchdog-based reliability and stop behavior

This project demonstrates how perception, control, and safety layers interact in a structured robot system.
